Qu'est-ce que ce projet ?

Ce projet consistait simplement à faire un blog fonctionnel.

Sur la page d'accueil on retrouve les différents posts existant dans la base de données, un lien cliquable menant à une page admin ainsi qu'un bouton permettant de faire apparaitre un formulaire pour ajouter un post.

En cliquant sur le lien pour accéder à la page d'administration, on arrive sur une page de connexion afin d'arriver à la page.

Une fois sur la page d'administration, on retrouve les différents posts ainsi que 3 boutons pour ajouter, modifier ou supprimer un post.

Sur la page d'accueil, nous pouvons également cliquer sur le titre d'un poste pour voir son contenu complet, son auteur et la date de création.

Lorsque nous avons cliquer pour voir le contenu complet d'un post, nous pouvons ajouter un commentaire à celui-ci
